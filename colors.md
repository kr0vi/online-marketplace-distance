# Website Color Theme

This file documents the current storefront color system used across the frontend.

## Primary Colors

- Primary Brand: `#332518`
- Primary Hover/Darker: `#24190f`
- Primary Disabled: `#a99684`
- Primary Light Surface: `#f8f2ea`

## Secondary Colors

- Secondary Text: `#6d5a48`
- Secondary Text Muted: `#85715d`
- Body Text Dark: `#4f3d2c`
- Placeholder Text: `#9b8b7b`

## Background Colors

- App Background: `#FBFAF7`
- Global CSS Background Token: `#f8f7f4`
- Card Background: `#ffffff`
- Skeleton Background 1: `#eadfce`
- Skeleton Background 2: `#f1e8da`
- Empty/Soft Background: `#faf6f1`

## Border Colors

- Default Border: `#ded4c8`
- Soft Divider Border: `#e7dfd5`
- Dashed/Muted Border: `#d8cec3`
- Hover Border: `#bfa58e`
- Accent Border (Primary): `#332518`

## Feedback Colors

- Error Background: `#fff2ea`
- Error Border: `#d8b4a0`
- Error Text: `#7a2f08`

## Text Tokens (Global)

- Global Foreground Token: `#292929`

## Notes

- Main action buttons should use primary brand colors.
- Content cards should stay on white with default border for consistency.
- Error surfaces should use the dedicated feedback palette above.
